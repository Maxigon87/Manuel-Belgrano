🌐 Plataforma web escolar desarrollada con tecnologías modernas y desplegada en Cloudflare.
🚀 Descripción

Este proyecto es una página web institucional que evoluciona hacia una aplicación completa, permitiendo:

🌍 Información pública de la escuela
🔐 Acceso privado para usuarios (docentes / alumnos)
📂 Contenido exclusivo (materiales, horarios, etc.)
⚡ Alto rendimiento gracias a Cloudflare
🧠 Tecnologías utilizadas
🧱 HTML, CSS, JavaScript
⚙️ Cloudflare Pages (frontend)
🔌 Cloudflare Workers (backend/API)
🗄️ Cloudflare D1 (base de datos SQL)
